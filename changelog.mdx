## Recent Updates

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added live scoring dashboard for real-time judge inputs during competitions
- Introduced athlete profile import from CSV files for faster event setup
- Enabled custom scoring rubrics per discipline (pole, aerial silk, hoop)

## Improvements

- Optimized routine playback speed controls for smoother video review
- Enhanced mobile responsiveness on tablets for on-site judging

## Bug Fixes

- Fixed sync issues between judge devices and central scoreboard
- Resolved export errors for PDF score sheets with special characters

</Update>

## Upgrade Instructions

<Callout kind="info">
  Always back up your event data before upgrading. Check the [release notes](https://github.com/workflow-inspiration/releases) for detailed migration steps.
</Callout>

<Steps>
  <Step title="Download Latest" icon="download">
    Visit the [downloads page](https://dashboard.example.com/downloads) and get the installer for your platform.
  </Step>
  <Step title="Install" icon="package">
    <Tabs>
      <Tab title="macOS" icon="apple">
````bash
sudo installer -pkg WorkflowInspiration-2.1.0.pkg -target /
````
      </Tab>
      <Tab title="Windows" icon="monitor">
````bash
WorkflowInspiration-2.1.0.exe /S
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Migrate Data" icon="database">
    Launch the app and follow the built-in migration wizard to import your previous events.
  </Step>
  <Step title="Verify" icon="check-circle">
    Create a test event and simulate a full scoring flow to ensure everything works.
  </Step>
</Steps>
